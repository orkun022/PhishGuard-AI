# Veri Seti Bilgileri

## Demo Veri Seti
`raw/phishing_urls.csv` dosyası, projede kullanılan demo veri setidir.

## Gerçek Veri Setleri (Kaggle)
Daha büyük ve gerçekçi veri setleri için:

1. **Phishing Site URLs** — https://www.kaggle.com/datasets/siddharthkumar25/malicious-and-benign-urls
2. **Web Page Phishing Detection** — https://www.kaggle.com/datasets/shashwatwork/web-page-phishing-detection-dataset
3. **PhiUSIIL Phishing URL Dataset** — https://www.kaggle.com/datasets/akashkr/phiusiil-phishing-url-dataset

İndirdiğiniz veri setini `raw/` klasörüne koyun ve `src/preprocessing.py` dosyasındaki dosya yolunu güncelleyin.
